DESCRIPTION:

Your Job is a simple, clean, mobile-friendly website template that has all of the necessary components 
for an effective coming soon page. It contains a countdown timer, a about section where you can 
put information about your team, company, group, etc., a contact area, and much more. Furthermore, 
it is simple to customise and is built with clean, well-organized code.
